# 🌍 Abroadly — Study Abroad Platform

Abroadly is a modern React-based study abroad platform designed to help students explore global education opportunities, universities, and international career pathways through an interactive and premium user experience.

---

# ✨ Features

- 🌎 Explore top study abroad destinations
- 🏫 Browse globally ranked universities
- ❤️ Save universities to wishlist
- 🌙 Dark / Light mode support
- 🔍 Search countries instantly
- 📱 Fully responsive design
- ⚡ Fast performance with Vite
- 🎨 Premium glassmorphism UI
- 💾 Persistent wishlist using localStorage

---

# 🚀 Tech Stack

- React.js
- Vite
- React Router DOM
- Context API
- JavaScript (ES6+)
- CSS3

# ❤️ Wishlist CRUD Functionality

The project implements CRUD operations through the university wishlist system.

## Operations

- Create → Save university
- Read → View saved universities
- Delete → Remove saved university
